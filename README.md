# BentOS

The **Bento** **O**peration **S**ystem.

## Requirements

* Ruby 2.7
* PostgreSQL 12

## Setup

1. Run `bin/setup`.
2. Edit `.env`.
3. Edit `config/database.yml` and run `bin/rails db:setup` if needed.

## Deploy

### Heroku

This app is deployable to [Heroku](https://www.heroku.com/).
